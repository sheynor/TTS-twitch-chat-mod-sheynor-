Для того чтобы запустить ваше приложение, вам понадобятся следующие библиотеки:

pip install asyncio - для асинхронной работы с сокетами и чатом.

pip install edge_tts - для синтеза речи через Microsoft Edge.

pip install pygame - для воспроизведения звука (MP3).

pip install tkinter - для создания графического интерфейса с полями ввода.

pip install os - для работы с файловой системой (создание и удаление файлов).

pip install time - для создания уникальных имен файлов (на основе времени).
